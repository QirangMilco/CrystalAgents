# CrystalAgents

## 原仓库
- [lukilabs/craft-agents-oss](https://github.com/lukilabs/craft-agents-oss)

## 新增 Feature
- 标题生成增强：支持 `miniModel`、多语言优化、标题校验、失败回退与调试能力。
- 会话记录导航：支持在会话中快速定位并跳转到特定消息。
- 旧版数据迁移与导入：支持检测旧版工作区数据，并提供手动迁移与导入入口。
- 应用变体配置与构建：支持自定义应用标识、名称、配置目录与变体构建流程。
- 工作区运行时数据目录重构：将运行时数据迁移至 `.craft-agents` 目录，并支持自动迁移与备份。
- 专注模式增强：支持悬浮侧栏延迟设置与活动栏容器高度选项。
- 桌面构建与 CI 增强：新增桌面构建工作流，并优化多平台构建与产物上传流程。
